Рязанский Perl воркшоп 2015

«Рязанский Perl воркшоп 2015» — первая в Рязани тематическая конференция, посвященная языку программирования Perl. Мероприятие пройдёт 15-16 августа 2015 года.

20 декабря впервые в рамках Saint Perl будет проведён хакатон, тема которого — «Высокоуровневый интерфейс к Git из Perl». Git — отличная и многими любимая система контроля версий, но из-за его внутреннего устройства программная интеграция с ним нетривиальна. Решить эту проблему призвана библиотека libgit2.

Конференция пройдёт по адресу: Московское шоссе, 39, корп. 5, офис "EPAM Systems". Начало конференции 11:00, телефон для связи +7(911)7767108.

15 августа — день докладов. Если вам есть, что рассказать — ждём ваших заявок.

Расписание конференции

Кроме того, в этот раз на конференцию приедет гость — brian d foy, автор и соавтор классических книг по Perl (Learning Perl, Intermediate Perl, Mastering Perl, Programming Perl), автор множества CPAN-модулей, основатель Perl Mongers, обладатель награды White Camel и множества других регалий.
